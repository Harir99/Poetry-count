INTRODUCTION
------------
The program will allow the user to type in several lines of poem. The user has to signal (".") on a a newline to indicate
that their input has ended. After the end of the input stage, the program will tell the user of the input stage, the program
will tell the user how many lines and words are in the poem and how many words are on each line of the poem.
Note: You are allowed to enter 100 lines of poetry.

RUNNING THE PROGRAM
-------------------
TO run the program type the following on your terminal:
    gcc -ansi -o poetry poetry.c
    ./poetry
